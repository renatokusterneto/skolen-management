Pilares da Marca

Missão

Transformar vidas

Não ensinamos conteúdo — criamos caminhos. Cada aluno que conclui um curso da Skolen sai com uma conquista real, não apenas um certificado.

Posicionamento

Educação que chega

Acessível, relevante e aplicável. A Skolen existe para quem quer crescer de verdade — sem barreiras de linguagem, tempo ou dinheiro.

Tagline Principal

Aprenda. Cresça. Transforme.

Três verbos em progressão. Do conhecimento à conquista. Use em headlines de campanha e materiais de apresentação.

Tagline Alternativa

O lugar onde o aprendizado vira conquista.

Use em landing pages, materiais de vendas e peças de conversão. Foco no resultado, não no processo.

Personalidade

Psicologia das Cores

Skolen Blue — #1E4ADB

Confiança, conhecimento, profundidade. O azul é a cor do saber. Remete a clareza, inteligência e propósito. É a cor que comunica "você pode confiar em nós para aprender".

Skolen Amber — #F5A623

Energia, conquista, otimismo. O âmbar é a cor da realização. Aquece, motiva e acende o desejo de avançar. Representa o "brilho" do momento em que algo faz sentido.

Voz & Tom

A Skolen fala como um mentor experiente que trata o aluno como igual — nunca condescendente, sempre próximo e focado no que o aluno conquista.

✓ Use assim

"Você já sabe mais do que imagina. Agora é hora de organizar esse conhecimento e ir além."

"Em 4 semanas, você vai dominar o que leva anos para aprender no mercado."

"Não é sobre estudar mais — é sobre estudar certo."

✗ Evite assim

"Nossa plataforma utiliza metodologias pedagógicas inovadoras baseadas em frameworks educacionais."

"Adquira nosso produto e obtenha acesso ao conteúdo premium da nossa solução de e-learning."

"Clique aqui para se cadastrar e aproveitar nossas funcionalidades."

Conscientização

Tom inspirador e empático. Fale sobre o problema ou sonho do aluno — não sobre a Skolen. "Você merece crescer."

Consideração

Tom educativo e confiante. Mostre o caminho, não o produto. "Veja como funciona na prática."

Conversão

Tom direto e urgente. Foco na conquista específica. "Comece hoje. Sua próxima versão te espera."

Tipografia

Nunito para títulos — moderna, geométrica, com personalidade. Inter para corpo — máxima legibilidade em qualquer tamanho.

Dupla Tipográfica

Body — Inter Regular

Referência CSS Completa

Copie estes tokens para o arquivo design-tokens.css do seu projeto. Importe antes de qualquer outro CSS.

/* ============================================ SKOLEN DESIGN TOKENS — v1.0 skolen.com.br · 2026 ============================================ */ /* LAYER 1: PRIMITIVES */ :root { /* Brand Colors */ --color-blue-600: #1E4ADB; --color-amber-400: #F5A623; /* Typography */ --font-family-heading: 'Nunito', sans-serif; --font-family-body: 'Inter', sans-serif; /* Spacing (4px base) */ --space-4: 1rem; /* 16px */ --space-6: 1.5rem; /* 24px */ } /* LAYER 2: SEMANTIC */ :root { --color-primary: var(--color-blue-600); --color-accent: var(--color-amber-400); --color-background:var(--color-gray-50); } /* LAYER 3: COMPONENTS */ :root { --button-bg: var(--color-primary); --button-radius: var(--radius-xl); /* 12px */ --card-radius: var(--radius-2xl); /* 16px */ --input-radius: var(--radius-xl); /* 12px */ } /* DARK MODE */ .dark { --color-background: var(--color-gray-900); --color-foreground: var(--color-gray-50); }
